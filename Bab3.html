PERENCANAAN KEBUTUHAN DAN DESAIN SISTEM

1. Perencanaan Kebutuhan

1.1 Identifikasi Masalah

  1. Penyewaan mobil memiliki daftar harga yang bervariasi menurut jenis mobil dan waktu lamanya menyewa mobil tersebut.
  2. Saat ini calon penyewa masih harus datang langsung ke kantor atau menghubungi lewat telepon untuk menanyakan mobil yang tersedia, booking, hingga pembayaran langsung ditempat.
  3. Perhitungan denda yang masih manual.
  4. Pembuatan laporan pun masih dilakukan penghitungan dengan menggunakan Ms. Excel.

1.2 Kebutuhan Software

Dalam pembuatan sistem informasi penyewaan mobil ini, perangkat lunak yang digunakan adalah :

  1. Sistem operasi yang digunakan yaitu Microsoft Windows 8
  2. Web editor yang digunakan dalam membangun aplikasi yaitu Visual Studio Code
  3. Web server yang digunakan yaitu XAMPP Control Panel v3.2.2
  4. Database yang digunakan yaitu MySQL
  5. Web browser yang digunakan yaitu Google Chrome
  6. Bahasa Pemograman yang digunakan yaitu PHP
